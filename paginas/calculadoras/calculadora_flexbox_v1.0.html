Calculadora Estándar CSS flexbox.

Cuando se trataron los temas CSS flexbox y CSS grid, me parecieron dos formas diferentes de abordar la misma tarea y una alternativa a CSS position, por lo que recordando el proyecto de la calculadora que había diseñado con CSS position y con la intención de conocer más estas nuevas alternativas, decidí repetir el ejercicio utilizando ambos métodos.

Primero lo hice empleando CSS flexbox y rápidamente me di cuenta que, por lo menos, en proyectos de ese tipo, el código se simplifica muchísimo y se puede obtener un resultado casi idéntico al original.

Por otro lado, tuve un inconveniente, ya que necesite hacer ligeras modificaciones al código html, por qué no logre la misma disposición de botones que en el proyecto original, no sé si esto es por falta de conocimientos o si realmente eran necesarias esas modificaciones.

Este es el resultado de la calculadora creada con CSS flexbox, el cual me parecio un método muy interesante.
