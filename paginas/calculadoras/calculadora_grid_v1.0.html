Calculadora Estándar CSS grid.

Para concluir el tema de la calculadora, realicé el proyecto empleando CSS grid y me sorprendió lo fácil que fue realizar el proyecto con este método, ya que la cantidad de código fue incluso ligeramente menor que con CSS flexbox, además de que no tuve que realizar ningún cambio al código html y los botones de la calculadora quedaron con la misma disposición que en el proyecto original.

El único inconveniente que tuve al realizar el proyecto de la calculadora con CSS grid, es que no encontré una forma de hacer que la propiedad --grid-column-gap-- permita que los espacios entre columnas sean de diferentes dimensiones, por lo que tuve que aplicar una solución que no me convenció del todo, pero siendo objetivo, creo que ese inconveniente era solo un capricho sin importancia, ya que se pudo solucionar fácilmente con un margen.

Este es el resultado de la calculadora creada con CSS grid, personalmente me gustó mucho este método e intentare aplicarlo en mis proyectos de certificación.
