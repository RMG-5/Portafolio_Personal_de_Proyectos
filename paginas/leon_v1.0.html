Diseño web receptivo heredado...

León CSS position.

Mientras realizaba el curso de Diseño web receptivo heredado, impartido por FreeCodeCamp, me pareció muy curioso ver la figura de un pingüino creada totalmente con CSS, confieso que estoy iniciando en el diseño web y la programación en general, pero me parece que CSS no fue concebido para un uso como ese, más aún si consideramos lo fácil que sería realizar un proyecto igual en un programa de diseño gráfico.

Por otro lado, me fascino la creatividad que tuvieron para utilizar CSS de esa manera, por lo que me inspiré y puse manos a la obra para realizar mi primera figura utilizando CSS position.

Por gusto personal, me pareció buena idea iniciar con un León, popularmente conocido como El Rey de la Selva, y aquí está el resultado.
